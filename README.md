# Taskora – Client Request Portal

## Overview
Taskora is a Django-based client portal where users can submit requests and communicate through a messaging system.

## Features
- User authentication (login/register/logout)
- Create and manage requests
- Role-based access (admin vs user)
- Messaging system per request
- Dashboard with request statistics
- Filtering by request status

## Tech Stack
- Django
- PostgreSQL
- Bootstrap

## How to run
1. Clone repo
2. Install requirements
3. Run migrations
4. Start server

## Notes
This project was built to practice backend development, authentication, and relational data handling.
